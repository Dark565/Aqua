## EXIT(2)

## NAME
	exit - Exit a process

## SYNOPSIS
```c
	#include <stdlib.h>

	void exit(int status);
```

## DESCRIPTION
	This syscall exits a process and frees its resources.

## RETURN VALUE
	Nothing is returned.

## ERRORS
	No errors
